# Sanae Moderation Bot

Sanae is Discord Moderation Bot, Using Discord.js V12

## Installation
Install Node.js [From Here](https://nodejs.org/en/).
Use NPM to install packages

fill settings.json with your information

npm i

## Usage

node bot.js
